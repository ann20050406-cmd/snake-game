# 🐍 Python 貪食蛇 (Snake Game)

## 專題簡介
這是一個使用 Python 製作的經典貪食蛇遊戲。玩家需要控制蛇的方向來吃掉食物，每吃一個食物身體就會變長。本專案是為了練習 Git 版本控制與 GitHub 協作流程而製作。

## 使用套件
* Python 3
* Turtle (Python 內建繪圖模組，無需額外安裝)
* Random (亂數模組)

## 遊戲玩法
1.  執行 `game.py` 檔案。
2.  使用鍵盤的 **「上下左右」方向鍵** 控制蛇的移動。
3.  吃到紅色食物：得分 +10，蛇身變長。
4.  **特色玩法 1**：隨著分數增加，蛇的顏色會改變！
5.  **特色玩法 2**：每吃 5 個食物，蛇的移動速度會變快！
6.  撞到牆壁或撞到自己的身體，遊戲結束。
