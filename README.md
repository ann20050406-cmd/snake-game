# 🐍 Python 貪食蛇 (Snake Game)

## 專題簡介
本專案是以 Python 程式語言結合 Turtle 繪圖模組所開發的經典「貪食蛇」遊戲。除了保留傳統吃食物、變長的核心玩法外，我們特別結合了程式邏輯設計，新增了視覺與難度上的雙重變化。此外，本專案全程使用 Git 進行版本控制，並透過 GitHub 實踐 Issue 追蹤與分支合併（Branch & Merge）的開發流程。

### 🌟 專案特色功能
為了增加遊戲的趣味性與挑戰性，我們在傳統玩法基礎上加入了以下兩大特色：

1.  **炫彩變色系統 (Chameleon Mode)**：
    * 突破傳統單調色彩，蛇頭與分數顯示會隨著每次吃到食物隨機改變顏色（如：青色、洋紅、黃色等），增加視覺豐富度。
    
2.  **極速暴走模式 (Rage Mode)**：
    * 加入動態難度調整機制。當玩家分數達到 **50 分** 時，遊戲背景將瞬間轉為 **深紅色** 警示，且蛇的移動速度會大幅提升，考驗玩家的反應神經！
      
## 使用套件
* Python 3
* Turtle (Python 內建繪圖模組，無需額外安裝)
* Random (亂數模組)

## 遊戲玩法
1.  執行 `game.py` 檔案。
2.  使用鍵盤的 **「上下左右」方向鍵** 控制蛇的移動。
3.  吃到紅色食物：得分 +10，蛇身變長。
4.  **特色玩法 1**：隨著分數增加，蛇的顏色會改變！
5.  **特色玩法 2**：每吃 5 個食物，蛇的移動速度會變快！
6.  撞到牆壁或撞到自己的身體，遊戲結束。
